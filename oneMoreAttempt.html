Cartograms with d3 & TopoJSON

This was our experimental cartogram (area based map). The math behind it is from this paper. We used d3.cartogram() which is a public library on github found by a google search here. There are a couple current issues with the cartogram. The main part is that the the algorithm doesn't work 100% correctly on our data set because the algorithm is for "continuous area", which is great for contiental United States, but not the world. Last we checked, North America hasn't been connected to the Europe for around 200 million years. Thus, we see that any country with value "1" has not disappeared off the map. However, there is sitll activity (espescially in continental Europe) which does show the principle behind a cartogram area expansion. Please wait for it to work (it is evaluating 240 choose 2 pairwise interactions for 35 iterations).
